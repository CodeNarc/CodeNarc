Also checks for ternary expressions where both expressions are the same constant or variable.
134UnnecessaryToStringChecks for unnecessary calls to toString().
135UnnecessaryTransientModifierThe field is marked as transient, but the class isn't Serializable, so marking it as transient has no effect.
136UnusedImportImports for a class that is never referenced within the source file is unnecessary.
137UseCollectManyIn many case collectMany() yields the same result as collect{}.flatten(). It is easier to understand and more clearly conveys the intent.
138UseCollectNestedInstead of nested collect{}-calls use collectNested{}
